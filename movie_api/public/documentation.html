This web application provides the user with information regarding movies based on comic books.
The user will be able to update their information and create a list of their favorite movies.

Request URL HTTP Method Query parameters Request body data format Response body data format
Get information about app /documentation GET None None None
Get data about all movies /movies GET None None JSON
Example:
{
"title": "Dick Tracy",
"director": "Warren Beatty"
},
Get data about a movie by movie title /movies/:title GET None None json
Get movies by genre name /movies/genres/:genreName GET None None json
Get movies by director's name /movies/directors/:directorName GET None None json
Allow new users to register /users POST None None json
Allow users to update their info by username /users/:Username PUT None None json
Allow users to add a movie to their favorites /users/:Username/movies/:MovieID POST None None json
Allows users to delete a movie from their favorites /users/:id/:movieTitle DELETE None None json
Allow an existing user to deregister /users/:Username DELETE None None json
